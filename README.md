## Features

- `product.json` 파일에서 fetch API 를 통해 비동기로 데이터를 받아옴
- 상품 추가
- 상품 삭제
- 상품 수량 수정
  - 1개 이하로 내려갈 시 버튼 비활성화
- 장바구니 초기화
- 총 가격 합산
- 장바구니 변동 시 localStorage 에 데이터 저장
  - 데이터가 있는 경우 렌더링 시 해당 데이터 보여줌
- 주문하기 클릭 시 주문완료 창 띄우며 장바구니 초기화
